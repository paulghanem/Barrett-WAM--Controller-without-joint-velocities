Virtual Robots Module Visualisation Add-in for Robotics Toolbox in MATLAB
-Developed as a part of RoboAnalyzer at Indian Institute of Technology Delhi, New Delhi, India by Prof. S.K. Saha and team
-Contact :  saha@mech.iitd.ac.in, roboanalyzer@gmail.com

Instructions:

1. Unzip the contents of the downloaded folder "VirtualRobotsModule" into the directory ~\robot-9.9\rvctools\robot\interfaces
2. If the MATLAB application is 32-bit, run the resigter32.bat file as an Administrator. If MATLAB installation is 64-bit, run the register64.bat file as an Administrator.). The command prompt windows pops up and displays a the message. Check the message to see if the application has been registered successfully, and press any key to exit
3. Add the folder "VirtualRobotsModule" to the MATLAB Path.
4. Add the Robotics Toolbox to the MATLAB path.
5. Run the example files to in the directory \robot-9.9\rvctools\robot\interfaces\VirtualRobotModule\Examples_IITD to see the capabilities of Virtual Robots module.


Note:
 - The MATLAB class VRM_Robot is placed in the directory \robot-9.9\rvctools\robot\interfaces\VirtualRobotModule\VRM_Robot
